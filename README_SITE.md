Site statique minimal pour le serveur Discord (SHADOW REAPERS — Blox Fruit)

Prévisualiser localement :

```bash
cd site
python3 -m http.server 8000
# puis ouvrir http://localhost:8000
```

Déploiement automatique :

Ce dépôt contient un workflow GitHub Actions (`.github/workflows/pages.yml`) qui publie automatiquement le contenu du dossier `/site` sur GitHub Pages à chaque push sur `main`.

La page principale du site est `shadow.html` (un `index.html` existant redirige vers `shadow.html`).

Après avoir poussé sur `main`, GitHub Pages prendra quelques minutes pour publier le site.
